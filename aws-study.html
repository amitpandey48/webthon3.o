About AWS Study Program

The AWS Study Program at Nirvana 2.0 is an intensive 3-day certification preparation course designed to equip students and professionals with comprehensive knowledge of Amazon Web Services. This program covers fundamental to advanced cloud computing concepts and prepares participants for AWS certification exams.

Led by AWS certified instructors and industry experts, the program combines theoretical learning with hands-on practice using real AWS environments. Participants will work on practical projects, complete labs, and receive guidance on exam strategies to maximize their certification success.

Certification Tracks

AWS Cloud Practitioner

Level: Foundational | Duration: 1 Day

Topics Covered:
  • Cloud computing concepts and AWS global infrastructure
  • Core AWS services and pricing models
  • Security and compliance fundamentals
  • Billing and support plans

Target Audience: Beginners with no cloud experience

Solutions Architect Associate

Level: Associate | Duration: 2 Days

Topics Covered:
  • Designing resilient and scalable architectures
  • EC2, VPC, S3, RDS, and other core services
  • Security best practices and IAM
  • Cost optimization and performance efficiency
  • Disaster recovery and backup strategies

Target Audience: Those with basic AWS knowledge

Developer Associate

Level: Associate | Duration: 2 Days

Topics Covered:
  • Developing and deploying applications on AWS
  • Lambda, API Gateway, DynamoDB
  • CI/CD pipelines with AWS services
  • Monitoring and troubleshooting
  • Security implementation in applications

Target Audience: Developers with AWS experience
